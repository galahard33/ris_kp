# GaussWebApp — Веб-интерфейс для решения СЛАУ методом Гаусса

Это веб-приложение на **ASP.NET Core MVC (C#)** позволяет:

- 🧮 **Генерировать** случайные невырожденные матрицы заданного размера (с диагональным преобладанием),
- 💾 **Скачивать** сгенерированную матрицу в виде текстового файла,
- 📤 **Загружать** собственные матрицы,
- ⚡ **Решать СЛАУ** методом Гаусса в **1, 2, 4 или 8 потоков**,
- 📊 **Сравнивать производительность** и точность решений,
- 👁️ **Просматривать** фрагменты матрицы и решения в удобном интерфейсе.
